    Tool Documentation

    Example for code/tools/hpc-clusters.md:

    NERSC (Perlmutter) - Login:

    ssh username@perlmutter.nersc.gov

    Job Submission:

    sbatch job.slurm
squeue -u $USER
scancel JOBID

    Sample SLURM Script:

    #!/bin/bash
#SBATCH -A m1234
#SBATCH -C cpu
#SBATCH -q regular
#SBATCH -t 01:00:00
#SBATCH -N 1

srun python my_script.py
