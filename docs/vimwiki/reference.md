# Reference

Quick reference for keybindings and maintenance schedules.

---

## Keybindings

### Navigation

| Keys | Action |
|------|--------|
| `<Leader>ww` | Open wiki index |
| `<Leader>w<Leader>w` | Open today's diary |
| `<Leader>ws` | Select wiki (if multiple) |
| `<Enter>` | Follow/create link |
| `<Backspace>` | Go back |
| `<Tab>` | Next link in page |
| `<Shift-Tab>` | Previous link |

### File Management

| Keys | Action |
|------|--------|
| `<Leader>wd` | Delete current wiki page |
| `<Leader>wr` | Rename current wiki page |

### Diary

| Keys | Action |
|------|--------|
| `<Leader>wi` | Diary index |
| `<Leader>w<Leader>i` | Generate diary links |

### Custom Bindings

| Keys | Action |
|------|--------|
| `<Leader>wi` | Open inbox |
| `<Leader>wt` | Open todo |
| `<Leader>wc` | Open school index |
| `<Leader>wr` | Open research index |
| `<Leader>wf` | Search wiki |
| `<Leader>ts` | Insert timestamp |

### Checkboxes

| Keys | Action |
|------|--------|
| `<C-Space>` | Toggle checkbox |
| `gl<Space>` | Remove checkbox |
| `gln` | Next checkbox state |

### Lists

| Keys | Action |
|------|--------|
| `<C-t>` | Increase indent |
| `<C-d>` | Decrease indent |

## Maintenance Schedules

### Daily (5 min)

- [ ] Process inbox
- [ ] Update diary
- [ ] Review today's tasks

### Weekly (30 min, Sunday)

- [ ] Generate diary index: `<Leader>w<Leader>i`
- [ ] Review diary entries
- [ ] Process all inboxes
- [ ] Update todo.md
- [ ] Review each active project
- [ ] Plan top 3 priorities for next week

### Monthly (1 hour, 1st of month)

- [ ] Review someday/maybe list
- [ ] Archive completed projects
- [ ] Review and update templates
- [ ] Check for orphaned notes (no incoming links)
- [ ] Update reading list

### Semester End

- [ ] Archive course notes
- [ ] Extract key concepts to research/
- [ ] Update thesis progress
- [ ] Review what worked, what didn't

### Yearly

- [ ] Archive old diary entries
- [ ] Review and prune reference/
- [ ] Major wiki reorganization if needed
- [ ] Set yearly goals
