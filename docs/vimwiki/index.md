# Vimwiki Power User Guide

*A comprehensive guide to using Vimwiki for maximum productivity*

This guide covers my complete Vimwiki setup for graduate research, GTD task management, and life organization. It's opinionated and tailored to my workflow as a physics graduate student, but the principles apply broadly.

## What's Covered

| Section | Description |
|---------|-------------|
| [Setup](setup.md) | Directory structure and Vim configuration |
| [LaTeX Snippets](snippets.md) | UltiSnips for math and physics |
| [Templates](templates.md) | Note templates for various use cases |
| [Daily Workflow](workflow.md) | Morning, throughout-day, and evening routines |
| [Research Workflow](research.md) | Zettelkasten for theoretical & computational research |
| [GTD System](gtd.md) | Inbox, next actions, someday/maybe |
| [Extras](extras.md) | Code notes, reading logs, hobbies |
| [Reference](reference.md) | Keybindings and maintenance schedules |

## Quick Start

1. Create the directory structure ([Setup](setup.md))
2. Add Vim configuration
3. Create templates folder
4. Start using the daily diary
5. Process inbox daily
6. Weekly review on Sundays
