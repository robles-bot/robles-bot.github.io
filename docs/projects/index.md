## US-ATLAS Tier-3 Benchmarking

**Organization:** US-ATLAS (2024–2026)

Developed benchmarking infrastructure for ATLAS Tier-3 Analysis Facilities at BNL, SLAC, NERSC, and University of Chicago. Built automated monitoring pipelines using Kubernetes, GitHub Actions, Elasticsearch, and Kibana.

**Topics:** HPC, Kubernetes, Elasticsearch, ATLAS computing, Python

## Higgs Mass Reconstruction

**Course:** PHYS 152 - Machine Learning in Physics (Winter 2024)

A neural network regression model to reconstruct the invariant mass of the Higgs boson from simulated H→μ⁺μ⁻ decay events. Uses PyTorch to learn the non-linear relationship between collider coordinates and particle mass.

**Topics:** Relativistic kinematics, collider coordinates, neural networks, regression

[View Project →](higgs.md)

## Double Pendulum Dynamics

**Course:** PHYS 115 - Computational Physics (Fall 2023)

Numerical simulation and analysis of double pendulum dynamics using Runge-Kutta methods. Explores chaotic motion, energy conservation, and normal mode analysis.

**Topics:** ODEs, Runge-Kutta integration, Lagrangian mechanics, chaos theory

## PYAR Outreach Curriculum

**Program:** PYAR/CrEST Outreach

A series of Jupyter notebooks introducing high school and early undergraduate students to particle physics data analysis. Students progress from Python basics to analyzing real CERN Open Data and discovering the Z boson.

**Topics:** Python, Pandas, collider coordinates, four-vectors, invariant mass, CERN Open Data
